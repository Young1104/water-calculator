## 1. 需求背景（精简）

水处理设计依赖多款互不连通的软件：一套管网设计含 22+ 步骤、平均每步用 3 款软件，4.3km² 管网需 388+ 次手工录入、生成 27+ 份文件。后果是重复录入、无法实时预览、反复纠错，且新人方案常因忽略现实因素而无法运行。Water Calculator 用「基于步骤的一体化计算 + 实时模拟 + 国标参数化 + 行业经验 AI 洞察」解决上述问题。

## 2. 服务用户与场景

| 用户 | 特征 | 关键场景 |
|---|---|---|
| 新人工程师（主） | 仅有理论、缺整体视角 | 合规计算、看懂模拟、获取经验建议 |
| 资深工程师 | 凭经验、面对新技术困惑 | 灵活调参、极端工况测试 |
| 指导老师/专家 | 关注落地性 | 评审、沉淀经验 |

三大核心能力：**①合规管理 ②实时模拟 ③行业洞察**，外加 **④项目协作工作台**。

> **重要结构决策**：合规管理、实时模拟、行业洞察 **不是独立的导航 Tab**，而是「设计文档」内的三种能力，以 Grammarly 式行内标注 + 文档右上「审阅」镜头呈现。设计文档不在侧栏出现，而是从项目卡片进入。

## 4. 全局界面框架

整体为「左侧栏 + 顶部栏 + 内容区」。进入设计文档时，内容区切换为「进度面板 + 显示面板 + 模拟面板」三栏工作区。

### 4.1 左侧栏 Sidebar
- **浅色**（Notion/Manus 风，非深色）。顶部产品 Logo（蓝色水滴 + 水务计算）。
- 一级导航：工作台、项目管理、工作模板、组件库、个人信息。
- 「最近项目」快捷区：彩色状态点 + 项目名，点击直达设计文档。
- 底部为当前用户（头像 + 姓名 + 等级），点击进入个人信息。
- 选中态：浅灰底 + 文字加粗（不使用重色块）。

### 4.2 顶部栏 Top Bar
- 左侧**面包屑**：常规页面显示页名；进入文档显示 `项目管理 › 进行中项目 › F50001 给水管网设计`，其中「项目管理 / 进行中项目」**可点击返回**（因此文档内不再单独设「返回」按钮，避免与面包屑重复）。
- 右侧：**搜索**（聚焦/输入弹出结果下拉，分类：项目/国标/工艺/组件）→ **协作者头像组**（点击弹出协作者面板 + 邀请）→ **通知铃**（点击弹出通知面板，非 Toast）→ **帮助**（点击弹出菜单：新手引导/国标库说明/快捷键/联系支持）→ **当前用户**（点击弹出用户菜单：个人信息/偏好设置/退出登录）。

### 4.3 三栏工作区（设计文档）
1. **进度面板（左）Progress Panel**：文档大纲式 Stepper，列出各级标题/步骤（用水量设计 → 节点流量计算[节点流量/管径/节点水压/水泵选型] → 流量分配 → 水力计算 → 计算校核 → 成果输出）。状态：✅已完成、🔵进行中、⚪未开始、🔺红色警告。**点击任意标题平滑锚定回正文对应位置，并对目标标题做一次蓝色焦点闪烁**；子项右侧小圆点标示该节存在合规/洞察问题。
2. **显示面板（中）Display Panel**：富文本计算文档。顶部有**行内标注图例**；正文含公式块、术语提示、国标关键词卡、内联计算「异形卡片」、Grammarly 式问题下划线。
3. **模拟面板（右）Simulation Panel**：标题「管网流量图（Linear Flow）」，浅色卡片实时渲染管网压力分布；问题管段橙色高亮、不利点红色、达标转绿。含关键指标（最不利点水压/设计流量/最大流速/环闭合差）与图例。

## 5. 核心模块产品与交互逻辑

### 5.1 工作台 Workspace
- **问候区**：`早上好，佟雪 👋 祝你有美好的一天` + 日期 + 待办/消息提示。
- **指标条**（4 张，可点下钻）：进行中项目、待我处理（→项目管理）、本周完成、合规通过率（→文档）。
- **进行中项目**：**卡片视图 / 列表视图可切换**。每张卡含：状态点、更新时间、负责人头像、项目名、描述、权限（Authority）、完成度进度条；点击进入设计文档。
- **执行中程序 Executing Program**：右栏列出正在跑的计算任务（带旋转动画、进度，如「环状管网平差 72%」「需氧量重算 · 等待中」）。
- **消息 Messages**：通知流（合规触发、项目创建、行业洞察、评审请求），带头像与时间戳。
- **临近节点**：按日期排列的待办（红/黄/蓝优先级点）。
- **效率看板**：软件切换 1 个平台、手工录入 ↓83%、中间文件 27→1（只读 KPI + 悬停 tooltip）。
- **新建项目**：点击打开**模板弹窗**——左侧「模板名称」可勾选列表，右侧「模板概览」（说明 + 模板功能图标组 + 成果预览缩略图），右上「保存」→ 创建并进入文档。

### 5.2 其他一级页面
- **项目管理**：项目卡片网格（编号/描述/状态点/权限/进度条/负责人/更多菜单）。顶部状态筛选 Tab（全部/计算中/评审中/等待中/已交付）**真实过滤**。
- **工作模板**：基于步骤流程的模板卡（给水管网/A²O/MBR/IC 厌氧/雨污分流/空白），点击打开新建弹窗并预选。
- **组件库**：可复用计算组件行（内置国标约束、专业标签），点击弹出**预览浮层**（说明+标签+「插入到文档」）。
- **个人信息**：资料卡 + 成长进度（经验点/等级/统计）+ 偏好设置（默认模板=下拉菜单；国标自动校验/行业洞察推送=开关 Switch）+「编辑资料」模态。

### 5.3 设计文档 · 文档内三大能力（行内标注体系）

文档正文用**四种行内标注**承载能力，配合右上「审阅」镜头（全部/合规/模拟/洞察）聚焦——选中某能力时，仅该类标记保持高亮，其余淡化。

| 标注 | 样式 | 触发 | 行为 |
|---|---|---|---|
| 国标关键词（合规） | 蓝色**实下划线** + ▾ | 点击 | **就地展开计算卡**：定义 + 公式 + 带 min/max 的参数输入（**越界即时变红报警**）+ 溯源链接 |
| 术语 | 蓝色**虚下划线** | 悬停 | 浮出释义 tooltip |
| 核心错误（模拟） | 红色**波浪线** | 点击 | 弹层：**列出全部连带错误**（如 H₃₈ −35m…）+ 建议替换（如 H=46m）+ **一键修复**；修复后下划线转绿、进度面板该步 🔺→✅、右侧模拟橙色高亮消失、不利点转绿、指标重算 |
| 行业洞察 | 紫色**波浪线** | 点击 | 弹层：基于真实根因**推荐设备 85A-10A** + 三维泵体图 + CAD 工程图 + 标签（高成本/需更多设备）+ 根因占比条 + 「采纳并填写理由」（沉淀经验闭环） |

**场景一 · 合规管理**：把"查国标"从离线翻文档变为计算中就地调用——关键词识别 → 缩略/就地展开 → 国标参数化（取值范围约束 + 越界提示）→ 溯源出处。

**场景二 · 实时模拟**：核心错误引发连带错误 → 弹层列依赖 → 一键修复 → 右侧管网图实时刷新。状态：正常/警告(黄)/错误(红+依赖)/修复(绿)。中部「水力计算」内联计算卡可调流量/流速/扬程/节点水压滑块，**实时联动右侧模拟**。

**场景三 · 行业洞察**：报告错误 → 专家建议 + Update → 行业洞察卡（推荐设备 + 3D/CAD + 标签 + 根因统计）→ 采纳并填写操作理由 → 系统聚合失败根因展示给后来者。

### 5.4 设计文档 · Notion 式文档工具栏与编辑交互

- **顶部文档栏（瘦身）**：左侧 插入 / 撤销 / 重做 + **「已保存 · 刚刚」**状态；中部留白；右侧 **审阅镜头** + 评论 / 历史版本 / 收藏 / 分享 / 更多。不再常驻格式按钮行。
- **选区气泡工具栏**：在正文**选中文字**才浮现格式条（转为▾ / B I U S / 行内代码 / 颜色 / 链接 / 评论），随选区定位、滚动即隐——Notion 标志性交互。
- **块悬停手柄**：鼠标移到任意段落/标题，左侧浮现 `＋`（插入内容）与 `⋮⋮`（块操作：转为/复制/评论/删除）。
- **行尾 slash 提示**：`输入 / 唤起命令，或选中文字以设置格式…`，点击唤起插入菜单。
- **全宽显示**：更多菜单内开关，切换正文标准宽度 ↔ 全宽。
- **历史版本**：面板列出版本 + 编辑者 + 时间，可「恢复到该版本」。
- **更多菜单**：复制链接 / 复制副本 / 移动到 / 导出成果 / 全宽显示 / 删除。

## 6. 交互反馈范式（统一规范）

不同操作必须使用与其性质匹配的反馈，**不得一律用 Toast**：

| 性质 | 模式 | 适用 |
|---|---|---|
| 已完成的动作确认 | **Toast**（底部居中、自动消失） | 撤销/重做、保存、导出、删除/复制/归档、一键修复、采纳建议、复制链接等 |
| 浏览内容/列表 | **面板 Popover** | 通知、协作者、历史版本、搜索结果、组件预览 |
| 选择一组操作 | **菜单 Menu** | 帮助、用户菜单、更多菜单、插入、文本样式、颜色、块操作、项目三点菜单 |
| 开/关设置 | **开关 Switch** | 国标自动校验、行业洞察推送、全宽显示 |
| 编辑/录入 | **模态 Modal** | 新建项目、编辑资料 |
| 文字格式 | **选区气泡** | 选中文字后浮现 |

浮层系统统一：自动避让视口边缘、点击外部关闭；菜单项统一样式（图标+文字，危险项红色）。

## 7. 关键交互流程（端到端）

1. 登录 → 工作台查看进行中项目/消息/执行中程序 → 点击项目卡或新建项目（选模板 → 预览 → 保存）。
2. 进入三栏工作区，按进度面板步骤逐步计算；点大纲标题锚定 + 焦点闪烁。
3. 输入参数 → 国标关键词就地展开、参数受取值范围约束、越界报警（场景一）。
4. 出现核心错误 → 弹层列连带依赖 → 一键修复 → 右侧模拟实时刷新、进度面板状态更新（场景二）。
5. 决策有落地风险 → 行业洞察卡给出设备/根因建议 → 采纳并填写理由（场景三）。
6. 编辑时选中文字浮现气泡、悬停块浮现 ＋/⋮⋮、`/` 唤起插入。
7. 完成 → 成果输出导出（平差表/水压表/水泵清单/CAD 数据）→ 工作台同步进度与消息给协作者。

## 8. 视觉规范

- **风格**：Notion / Manus 极简——浅色、暖中性、极淡边框、几乎无阴影、留白充足。
- **主色**：蓝 `#2D6BFF`（强调/链接）；主按钮使用更深的 `#1D52E0` 以保证白字对比度达标。
- **语义色**：错误红 `#E5484D`、完成绿 `#30A46C`、警告橙（模拟高亮）、洞察紫 `#9065D9`。
- **中性色（文字/边框）**：正文 `#37352F`、次级 `#6B6A66`、三级 `#737269`、边框 `#EDEDEB`、悬停底 `#F3F2F0`、侧栏底 `#FBFBFA`。
- **可访问性（WCAG 2.1 AA）**：所有正文/次要文字对比度 ≥ 4.5:1（次级 #6B6A66 ≈ 5.4:1、三级 #737269 ≈ 4.8:1）；主按钮白字 on #1D52E0 ≈ 6.3:1。只读 KPI 不做成按钮。
- **字体**：系统无衬线；公式与数值用等宽字体（含上下标、根号）。
- **行内标注图例**（文档顶部固定）：蓝实线=国标可展开、蓝虚线=术语释义、红波浪=核心错误、紫波浪=行业洞察。
- **数据可视化**：管网流量图（Linear Flow）为视觉核心，问题段橙色、不利点红色、达标绿色。

国标编号（GB 50013、GB 50014、CJJ 60 等）、设备型号（CK100/20 H、85A-10A）、工艺缩写（A²/O、MBR、IC）、公式符号（q_mi、Q_h、H₁ 等）、单位（m、L/s、km）、等级（Lv.3）。

## 9. 成功标准

- **效率**：软件切换从平均每步 3 款降为 1 个平台；重复录入相对 388 次基线下降 80%+；中间文件与跨软件传递明显减少。
- **质量**：国标问题在计算阶段被约束拦截，事后纠错轮次下降；"系统无法运行"的方案比例下降。
- **体验**：新人能依模拟独立完成迭代，认可"实时预览有助决策"；行业洞察建议采纳率与据其修正成功率。

## 10. 设计迭代变更记录

1. **视觉风格**：由工程蓝/深色侧栏 → shadcn → 最终采用 **Notion/Manus 浅色极简**（用户明确选择浅色侧栏，覆盖早期"深色近黑侧栏"设定）。
2. **结构收敛**：合规/模拟/洞察由独立 Tab → 收敛为**设计文档内的能力**（行内标注 + 审阅镜头）；文档从项目卡片进入。
3. **统一文档工作区**：计算/合规/模拟/洞察集中到一份连续文档；左侧进度面板=可锚定的文档大纲；右侧管网流量图实时联动内联计算卡。
4. **Grammarly 式标注**：四类行内标记（国标/术语/核心错误/洞察），点击就地展开或弹层，文档顶部加图例。
5. **工作台增强**：问候、指标条、卡片/列表切换、执行中程序、消息、临近节点、效率看板、新建项目模板弹窗。
6. **全文中文化**：界面英文文案全部改中文（保留国标号/型号/工艺缩写/公式符号/单位）。
7. **全量可交互**：所有卡片/按钮/图标接上行为；引入 Toast 反馈系统。
8. **反馈范式纠偏**：把"内容/设置型"误用的 Toast 改为面板/菜单/开关/模态（通知→面板、协作者→面板、用户/帮助/三点→菜单、偏好→开关、编辑→模态、搜索→结果下拉）。
9. **可访问性**：次要文字与主按钮对比度修复至 WCAG AA；只读 KPI 不再做成按钮。
