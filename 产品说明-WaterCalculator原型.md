## 一、产品概述

水处理计算平台把分散在多款软件中的水处理工程计算，整合到一个"基于步骤"的一体化平台，核心价值是**计算、合规、模拟、经验四件事在同一份文档里完成**：

- **一体化计算**：以步骤（STEP）为主线组织全流程，消除跨软件跳转与重复录入。
- **国标参数化**：把可查阅的国家标准转成计算中的取值范围约束，输入越界即时提示。
- **实时模拟**：调整参数时，右侧管网流量图实时刷新，问题环节高亮，所见即所得。
- **行业洞察**：沉淀真实项目的失败根因与专家经验，向新人推荐更可落地的决策。

目标用户：新人工程师（主）、资深工程师、指导老师/行业专家。

## 二、信息架构与导航

侧边栏为五个一级入口，外加"最近项目"快捷区：

| 入口 | 作用 |
|---|---|
| 工作台 | 个人首页：问候、指标、进行中项目、执行中程序、消息、临近节点、效率看板 |
| 项目管理 | 全部项目的卡片网格，支持状态筛选；点击进入设计文档 |
| 工作模板 | 基于步骤流程的项目模板库，一键新建标准化项目 |
| 组件库 | 可复用计算组件（内置国标约束与模拟），点击插入文档 |
| 个人信息 | 资料、成长进度、偏好设置 |

**关键结构变化**：合规管理、实时模拟、行业洞察**不再是独立的导航 Tab**，而是"设计文档"内的三种能力（审阅层）。设计文档不在侧栏出现，而是从项目卡片进入。

## 三、全局框架

- **侧边栏**：浅色（Notion/Manus 极简风），五个入口 + 最近项目 + 底部当前用户。
- **顶部栏**：面包屑（进入文档时显示"项目管理 › 进行中项目 › F50001 给水管网设计"）、全局搜索、协作者头像组、通知铃、帮助、当前用户。
- **设计文档为三栏工作区**：
  1. **进度面板（左）**：文档大纲式的步骤树（Stepper），状态含 ✅完成 / 🔵进行中 / ⚪未开始 / 🔺警告；点击任意标题**锚定**回正文对应位置。
  2. **显示面板（中）**：富文本计算文档，含公式块、术语提示、国标关键词卡、内联计算卡、Grammarly 式问题下划线。
  3. **模拟面板（右）**："管网流量图"，实时渲染管网压力分布，问题段橙色高亮、不利点红色。

## 四、逐屏功能说明

### 4.1 工作台
- **问候区**：早上好 + 日期 + 待办/消息提示。
- **指标条**（4 张，可点）：进行中项目、待我处理 → 跳项目管理；本周完成 → 提示；合规通过率 → 进文档。
- **进行中项目**：卡片/列表视图切换；每张含状态点、更新时间、负责人头像、权限、进度条；点击进入文档。
- **执行中程序**：正在跑的计算任务（带旋转动画、进度）；点击进入对应文档。
- **消息**：动态流（合规触发、项目创建、行业洞察、评审请求）；点击进入相关位置。
- **临近节点**：按日期排列的待办；点击进入文档。
- **效率看板**：软件切换、手工录入↓83%、中间文件 27→1；点击查看对标说明。

### 4.2 项目管理
- 顶部状态筛选（全部/计算中/评审中/等待中/已交付）**真实过滤**卡片网格。
- 每张项目卡：编号、描述、状态点、权限、进度条、负责人、更多菜单（重命名/复制/归档）。
- "新建项目"打开模板弹窗。

### 4.3 工作模板
- 六张模板卡（给水管网、A²/O、MBR、IC 厌氧、雨污分流、空白）；点击任意模板打开新建弹窗并预选该模板。

### 4.4 组件库
- 六个可复用计算组件，带专业标签（给水/污水/国标号/实时模拟/行业洞察）；点击"插入当前文档"。

### 4.5 个人信息
- 资料卡 + 成长进度（经验点/等级/统计）+ 偏好设置（默认模板、国标自动校验、洞察推送，均可点击）。

### 4.6 新建项目弹窗
- 左侧"模板名称"可勾选列表，右侧"模板概览"（说明 + 模板功能图标组 + 成果预览缩略图），右上"保存" → 创建项目并进入文档。

## 五、设计文档内三大能力（核心）

三种能力以 **Grammarly 式行内标注**呈现，在文档右上"审阅"开关可分别聚焦（全部/合规/模拟/洞察）。

### 5.1 合规管理（蓝色国标关键词）
- 文档中的国标专业名词（如"时变化系数""经济流速"）以蓝色高亮，**点击就地展开卡片**：
  - 名词定义 + 计算公式 + 带 min/max 边界的参数输入；
  - **越界即时提示**：输入超出国标取值范围，输入框变红并提示"越界！国标要求 x–y"；
  - 右上"溯源"链接，跳转规范出处（如 GB 50013 · 7.1.2）。

### 5.2 实时模拟（红色波浪下划线 = 核心错误）
- "节点水压 H₁ = 35 m"标红色波浪线，点击弹出建议浮层：
  - 列出**全部连带错误**（节点 38/22/17 水压、泵站扬程裕度…）；
  - "建议替换 35 m → 46 m"；
  - 点"一键修复"：下划线变绿、进度面板该步从 🔺 转 ✅、右侧管网图橙色高亮消失、不利点转绿、状态指标重算。
- 中部"水力计算"内联计算卡：调整流量/流速/扬程或拖动"节点水压"滑块，右侧模拟**实时联动**。

### 5.3 行业洞察（紫色波浪下划线）
- "CK100/20 H 泵"标紫色波浪线，点击弹出行业洞察卡：
  - 基于 214 个真实项目根因，**推荐设备 85A-10A**；
  - 三维泵体图 + CAD 工程图 + 标签（高成本 / 需更多设备）；
  - 根因占比条（成本偏高 80% / 配套偏多 45%）；
  - 点"采纳并填写理由"，沉淀为团队经验。

## 七、视觉规范

- **风格**：Notion / Manus 极简——浅色、暖中性、极淡边框、几乎无阴影、留白充足。
- **主色**：蓝 `#2D6BFF`（强调与链接）。
- **语义色**：错误红 `#E5484D`、完成绿 `#30A46C`、警告橙（模拟高亮）、洞察紫 `#9065D9`。
- **字体**：系统无衬线；公式与数值用等宽字体（含上下标）。
- **数据可视化**：管网流量图为视觉核心，问题段橙色、不利点红色。

## 八、与 PRD 的一致性对照

逐条核对详细版 PRD（共 22 个要点），均已实现：信息架构、三栏框架、Stepper 状态、工作台（卡片/列表、执行中程序、消息、新建项目模板弹窗）、合规关键词就地展开+取值范围越界提示+溯源、模拟核心错误+连带依赖列表+一键修复、行业洞察设备推荐+三维泵体/CAD+标签、顶栏协作者+面包屑+公式工具条、视觉配色。

**唯一有意偏离**：PRD §7 视觉规范写的是"深色近黑侧边栏"，本版按你的明确偏好保持 **Notion 浅色侧栏**。如需深色版，可一键切换。

**保留的非中文标识**（行业惯例，不属"英文文案"）：国标编号（GB 50013 等）、设备型号（CK100/20 H、85A-10A）、工艺缩写（A²/O、MBR、IC）、公式符号（q_mi、Q_h、H₁ 等）、单位（m、L/s）、等级（Lv.3）。

## 九、使用与后续建议

**建议体验路径**：工作台点项目卡 → 进入文档 → 点蓝色"经济流速"看就地展开并输入越界值 → 点红色"H₁=35m"看连带错误并一键修复（观察右侧管网与进度面板变化）→ 点紫色"CK100/20 H 泵"看设备推荐 → 拖动内联计算卡滑块看实时联动 → 返回项目管理用状态筛选。

**可继续优化**：①一键导出真实文件；②大纲滚动自动高亮（scroll-spy）；③深色侧栏切换开关；④更多国标关键词与组件；⑤多人协作光标。
